# 基础

:::note 提示

欢迎来到 初梦大陆服务器 ，初来乍到可能感觉手足无措、不知道怎么开始玩？往下阅读，开始初梦大陆之旅吧！

:::

安装好 Minecraft

- 请确保版本至少为Java 1.21.5 版本
- 手机版体验较差，不推荐作为主要游玩方式

## 进入服务器

恭喜你进入服务器啦🎉，开始时你可能会不知所措。请先设置自己的家（重生点）吧、指令是`/sethome jia`，然后开始开始建些什么吧！
